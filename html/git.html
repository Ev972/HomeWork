Git

  • git init - инициализации проекта как репозитория git.
  • git clone - клонирует репозиторий в новую директорию.
  • git status- для просмотра статуса файлов в локальном репозитории.
  • git add - добавляет новые или измененные файлы".
  • git commit -«закрепляет» промежуточные результаты.
  • git push -u origin -создаёт в удалённом репозитории ветку, соответствующую локальной и связывает их.
  • git branch --no-merged -увидеть все ветки, содержащие наработки, которые пока ещё не слиты в текущую ветку
  • .... И еще много интересных команд.
